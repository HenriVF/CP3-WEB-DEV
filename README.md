# 🎬 Lista de Filmes Assistidos - CRUD + Login

Este projeto foi desenvolvido como parte da atividade **CP3 - CRUD e IA Generativa** da FIAP.  
A aplicação é uma página web simples que simula um sistema de cadastro de filmes assistidos, com tela de login e operações de CRUD.

---

## 🚀 Funcionalidades

- **Tela de Login**
  - Usuário: `aluno`
  - Senha: `fiap2025`
  - Validação de campos obrigatórios e mensagens de erro visíveis.

- **CRUD Completo da Lista de Filmes**
  - Adicionar filme ao **final** da lista
  - Adicionar filme ao **início** da lista
  - Visualizar todos os filmes dinamicamente
  - Editar qualquer filme individualmente
  - Remover qualquer filme individualmente

- **Validações obrigatórias**
  - Não permite salvar itens vazios
  - Edição com campo vazio mantém o valor original
  - Remoção considera a posição do item na lista

---

## 🛠️ Tecnologias utilizadas

- **HTML5** para estrutura da página  
- **CSS3** para estilização com tema cinematográfico  
- **JavaScript puro** para lógica de login e CRUD  

---

## 🎨 Estilo visual

O design foi inspirado em pôsteres de cinema, com:
- Fundo escuro em gradiente (clima de sala de cinema)  
- Títulos em amarelo com sombra (referência a pôsteres clássicos)  
- Botões em amarelo e vermelho (cores vibrantes de bilheteria e tapete vermelho)  
- Efeitos de hover e sombras para dar dinamismo

## 🤖 Uso de Inteligência Artificial

Foram consultadas três IAs diferentes (ChatGPT, Copilot e Gemini) com o mesmo prompt.  
O arquivo `prompts.md` contém:
- Os prompts usados  
- Problemas encontrados em cada resposta  
- Justificativa da IA escolhida como base  

A solução final foi baseada no **Copilot**, com ajustes para mensagens de erro e estilo visual.
